Initialize TypeScript for Frontend:
cd frontend

npx tsc --init

**if no npm**:
** Install Node.js + npm (Recommended Method)
** The best way to install Node.js and npm is through NodeSource.

** Install the Latest LTS Version (Recommended for Stability)

** curl -fsSL https://deb.nodesource.com/setup_18.x | sudo bash -
** sudo apt install -y nodejs

** This will install:

** Node.js LTS (version 18)
** npm (Node Package Manager)
***
